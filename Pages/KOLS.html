How to integrate our code to yours

First Step: Get our unicate API key to make POST Requests

To get our unicate API Key you need type in your browser 89.203.251.114/get-key, then copy the Token value and paste it to txt file fot later usage.

Second Step: Make Json POST Request to our API Service

For json request you need create values with your OpenAI Input, Output, Max Tokens, Model, Temperature (Optional: top_p and n)
